# PythiaJS
Agnostic tool for user context detection and interpretation

## Getting started
Include PythiaJS into your HTML file(s)
```
<script type="text/javascript" src="pythia.min.js"></script>
```
Initialise the PyhtiaJS by calling the init method
```
pyhtia.init()
```
